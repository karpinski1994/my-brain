## Skill XP Formula

```
XP to rank up = Rank × BaseCost
Doing a quest = (Quest XP) × (Skill Rank × 0.1 + 1)
```

Example: Psychology Rank 3, complete a 50 XP psychology quest
→ Psychology XP earned = 50 × (3 × 0.1 + 1) = 50 × 1.3 = 65 XP

This means higher skill ranks accelerate further growth in that skill.
