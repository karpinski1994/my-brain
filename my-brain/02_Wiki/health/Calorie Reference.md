## Methodology

1. Weigh or estimate each food in **grams**
2. Look up **kcal/100g** from the table above
3. Apply: `kcal = (grams × kcal_per_100g) / 100`
4. Sum all items for the meal, then the day
5. Cross-check: stated total should match calculated sum within ~5%

> If a food isn't in this table, add it with its source. Never guess — use USDA or the package label.
